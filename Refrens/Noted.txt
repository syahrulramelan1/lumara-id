Gunakan pola MVC + OOP di semua layer:

MODEL LAYER (lib/models/):
  BaseModel.ts в†’ abstract class, method dasar:
    findById(), findAll(), create(), 
    update(), delete()
  
  Setiap model extends BaseModel:
    ProductModel.ts   в†’ override findAll() dengan filter
    UserModel.ts      в†’ findByEmail(), updateProfile()
    WishlistModel.ts  в†’ toggle(), isWishlisted()
    OrderModel.ts     в†’ findByUser(), updateStatus()
    ReviewModel.ts    в†’ findByProduct(), calcAvgRating()
    CategoryModel.ts  в†’ findBySlug()
  
  Semua model menggunakan Singleton pattern:
    export default new ProductModel()

SERVICE LAYER (lib/services/):
  Semua business logic ada di sini, bukan di komponen.
    ProductService.ts в†’ getProductList(), 
                        getProductDetail(),
                        formatPrice(),
                        calculateDiscount()
    WishlistService.ts в†’ getUserWishlist(),
                         toggleWishlist()
    AuthService.ts     в†’ login(), register(),
                         resetPassword()
    OrderService.ts    в†’ placeOrder(), 
                         getOrderHistory()

CONTROLLER LAYER (app/api/):
  API Routes sebagai controller.
  Hanya handle request/response.
  Semua logic didelegasi ke Service.
  
  app/api/produk/route.ts
  app/api/wishlist/route.ts
  app/api/auth/route.ts
  app/api/pesanan/route.ts

VIEW LAYER (app/ + components/):
  React Server Components untuk data fetching.
  React Client Components hanya jika interaktif.
  Tidak ada business logic di komponen.

OOP PRINCIPLES YANG HARUS ADA:
  Inheritance   в†’ semua Model extends BaseModel
  Encapsulation в†’ private method di Service
  Abstraction   в†’ BaseModel sembunyikan Prisma detail
  Polymorphism  в†’ override method di child Model
  Singleton     в†’ satu instance per Model/Service
